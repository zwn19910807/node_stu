# Node\.js \+ Express 后端接口学习项目

**✨ 基于 Node\.js \+ Express 的轻量级后端服务 \| 本地数据库 CRUD 接口实战项目**

这是一个面向学习实战的 Node 后端入门项目，基于 Express 框架快速搭建后端服务，对接本地数据库，封装完整的**增、删、改、查（CRUD）、上传下载**通用接口，代码简洁规范、注释清晰，非常适合新手学习 Node 后端基础、接口开发与数据库交互核心逻辑。

---

## 📦 技术栈

- **运行环境**：Node\.js

- **核心框架**：Express（轻量高效的 Node 后端框架）

- **数据存储**：本地 MySQL 数据库

- **核心能力**：传统简洁接口 API、数据库 CRUD、本地服务启动、接口调试

- **开发规范**：模块化拆分、统一接口返回格式、结构化代码分层

---

## 💻 环境准备

运行本项目前，请确保本地已安装以下环境：

1. **Node\.js**（v22\+ 推荐）

2. **npm / yarn** 包管理工具

3. **本地数据库**：MySQL（需本地安装并启动服务）

查看环境版本命令：

```Plain Text
node -v
npm -v
```

## 📥 项目安装与启动

### 1\. 克隆项目

```Plain Text
git clone https://github.com/zwn19910807/node_stu.git
cd node_stu
```

### 2\. 安装依赖

```Plain Text
npm install
```

### 3\. 数据库配置

修改项目中数据库配置文件（默认 `db.js`），替换为自己本地数据库的**账号、密码、数据库名**，确保本地数据库服务正常启动。

### 4\. 启动项目

```Plain Text
# 普通启动
npm start

# 热更新启动（开发推荐，需安装nodemon）
npm run dev
```

### 5\. 启动成功

项目默认运行在：`http://localhost:3000`（端口可在配置文件自定义修改）

## 📡 核心接口说明（CRUD）

项目采用**传统简洁接口设计**（非 RESTful 风格），基于 Express 原生 GET/POST 实现用户模块完整增删改查，适配新手入门开发，所有接口返回标准化 JSON 格式，调试简单、直观易懂。

|请求方式|接口地址|接口功能|备注说明|
|---|---|---|---|
|GET|/api/user/list|获取用户列表（查询全部数据）|支持批量查询，可扩展分页、条件筛选|
|POST|/api/user/add|新增用户数据|接收表单/JSON参数，新增数据库用户记录|
|POST|/api/user/delete|删除用户数据|传递用户ID，删除对应用户记录|
